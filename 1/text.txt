JavaScript - универсальный язык программирования, который используется для создания интерактивных веб-страниц.
Он отличается динамической типизацией и легкостью в изучении. Давайте начнем с основного синтаксиса JavaScript.

Объявление переменных
В JavaScript переменные объявляются с использованием ключевого слова let, const или var. Например:

let name = "Masha";
const age = 25;
var greeting = "Hello, world!";
Используя let мы создаем переменную, которая может быть переопределена.
При помощи const мы объявляем постоянную (константу), значение которой нельзя изменить.
Ключевое слово var также объявляет переменную, но имеет различное поведение, связанное с областью видимости.
 Обычно для объявления переменных рекомендуется использовать let или const

Динамическая типизация
JavaScript является языком с динамической типизацией, что означает, что тип переменной определяется во время выполнения,
 а не во время компиляции. Например:

let age = 25;  // Это число
age = "25";    // Теперь это строка
Это позволяет удобно работать с данными, но также требует внимательности при написании кода.

Названия переменных
При именовании переменных следует придерживаться определенных правил:

Имя переменной должно начинаться с буквы, символа подчеркивания (_) или знака доллара ($).
Последующие символы могут быть буквами, цифрами, подчеркиваниями или знаками доллара.
Имена переменных чувствительны к регистру (например, myVar и myvar - две разные переменные).
Константы
Константы объявляются с помощью ключевого слова const и представляют значения, которые не могут быть изменены после присвоения. Например:

const PI = 3.14;
Использование констант помогает предотвратить неожиданные изменения значений в коде.
